Overview

The SolutionMessage is responsible for receiving a PuzzleResponse from a completed puzzle and rendering the data into a string for display in the terminal.

def self.print(answer:, class_name:, date:, message:, type:)
  puts RESULT % {
    answer:  const_get("AdventOfCode::Days::#{class_name}").public_send(answer),
    date:    date.strftime('%B %d'),
    message: message,
    type:    TYPES[type]
  }
end
